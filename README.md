# img-recognition


## Objective
Train an image recognition model using Google Teachable Machine and classify input images using Python.

## Step 1: Create the Dataset
- Created two classes:
  - Cat
  - Dog
- Added training images for each class.

## Step 2: Train the Model
- Used Google Teachable Machine.
- Trained the model.
- Evaluated the accuracy.

## Step 3: Export the Model
- Exported the model as TensorFlow → Keras.
- Downloaded:
  - keras_model.h5
  - labels.txt

## Step 4: Python Script
- Loaded the model.
- Accepted an input image.
- Predicted the image class.

## Step 5: Output
The model successfully predicted the image class.
